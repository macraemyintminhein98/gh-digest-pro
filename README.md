# GH Digest Pro

**Your daily GitHub digest, simplified.**

GH Digest Pro is a lightweight web application that integrates with GitHub to provide a concise daily summary of open Pull Requests and Issues for your chosen repositories, delivered straight to your email. 

Stop digging through notifications and start your day with a clear overview of your GitHub activity. 

**Features:**
* Daily email digests of PRs and Issues.
* Select repositories to monitor.
* Secure GitHub OAuth integration.

Built for small development teams, individual open-source maintainers, and indie hackers.
